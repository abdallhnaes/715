# ديربي كفردريان V5

تطبيق أندرويد لإدارة خطط الفريقين ونتائج الديربي.

## الجديد في هذه النسخة
- اسم التطبيق أصبح: ديربي كفردريان.
- أيقونة جديدة للتطبيق.
- إضافة خطة يدوية: تستطيع وضع اللاعبين كما تريد بالسحب داخل الملعب.
- تستطيع استخدام الخطط الجاهزة: 3-3-1 / 3-2-2 / 2-3-2 / 1-3-3.
- إدخال خطة الفريق الأول والفريق الثاني.
- إدخال نتيجة المباراة بعد تجهيز الخطتين.
- عداد أعلى التطبيق لعدد انتصارات كل فريق والتعادلات.
- سجل كامل للمباريات.
